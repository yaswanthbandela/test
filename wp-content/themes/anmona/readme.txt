== Description ==
Anmona is Gutenberg based Full Site Editing WordPress block theme that satisfies different purposes. Effortlessly craft stunning websites with intuitive block-based editing. Customize every detail, seamlessly integrate plugins, and unleash limitless creativity.

== Installation ==
1. Go to Themes > Add New, search for "Anmona" and then click "Install".
2. After installaion is done click on "Activate".
4. Setup Site Logo and Navigation Menu.
5. Enjoy!
